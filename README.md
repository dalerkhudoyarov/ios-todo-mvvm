# ToDo List (MVVM)

Simple ToDo List iOS app built with MVVM architecture and local persistence.

## Features
- Add / delete tasks
- Mark task as done
- Local storage (UserDefaults)
- MVVM separation (View / ViewModel / Model)

## Tech Stack
- Swift
- UIKit
- MVVM
- UserDefaults
- Codable
